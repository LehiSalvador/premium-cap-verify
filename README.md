# Premium Cap Verify

Pagina estatica de verificacion NFC para gorras.

## Estructura

- index.html
- style.css
- script.js
- vercel.json
- assets/verification-seal.png

## Ejemplos de links

Con ID en la ruta:

https://premiumcapverify.com/SEC-D02-001

Con ID, nombre y scans en parametros:

https://premiumcapverify.com/?id=SEC-D02-001&name=ROCKSTAR%20ICT&scans=2

## Nota

El contador de scans es visual/estatico. Para contar scans reales se necesitaria una base de datos o analytics.
