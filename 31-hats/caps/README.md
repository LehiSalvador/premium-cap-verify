# 31 HATS - sublinks por gorra

Aqui puedes crear una carpeta por cada gorra especifica de esta marca.

Ejemplo:

31-hats/caps/nombre-del-modelo/index.html

La URL quedaria:

https://premiumcapverify.site/31-hats/caps/nombre-del-modelo/

Para cada gorra especifica cambia:
- El titulo de la pagina.
- El texto de class="cap-name".
- El data-scan-key para que cada gorra tenga su propio contador.
