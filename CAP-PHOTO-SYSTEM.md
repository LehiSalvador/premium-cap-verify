# Sistema de fotos por gorra

Cada marca tiene 20 espacios iniciales para gorras:

caps/001/
caps/002/
caps/003/
...
caps/020/

Dentro de cada carpeta hay:

- index.html
- name.txt
- INSTRUCCIONES.txt

Para cambiar el nombre que sale en la pagina:
Edita name.txt

Para cambiar la foto central:
Sube una imagen dentro de la carpeta correspondiente con el mismo nombre de la carpeta.

Ejemplo:

barbas-hats/caps/001/001.jpg

Tambien acepta:

001.png
001.webp
001.jpeg

Ejemplo de URL:

https://premiumcapverify.site/barbas-hats/caps/001/

Cada gorra tiene su propio contador porque cada una usa un data-scan-key diferente.
